# Data Structures and Algorithms (DSA)

A comprehensive collection of Data Structures and Algorithms implementations in C++ and JavaScript.

## 📚 Repository Structure

This repository contains implementations and examples of various data structures and algorithms organized into the following categories:

### Data Structures
- **[Array](./Array)** - Array manipulation and operations
- **[Linked List](./Linked_List)** - Singly, doubly, and circular linked list implementations
- **[Stack](./Stack)** - Stack data structure and applications
- **[Queue](./Queue)** - Queue implementations (linear, circular, priority queue)
- **[Binary Tree](./Binary_Tree)** - Binary tree operations and traversals
- **[BST](./BST)** - Binary Search Tree implementations and operations
- **[String](./String)** - String manipulation algorithms

### Algorithms
- **[Sorting Algorithms](./Sorthing%20Algorithms)** - Various sorting techniques (Bubble, Selection, Insertion, Merge, Quick, etc.)
- **[Binary and Decimal](./Binary%20and%20Decimal)** - Number system conversions

### Programming Concepts
- **[OOPs](./OOPs)** - Object-Oriented Programming concepts and examples
- **[JavaScript](./js)** - JavaScript implementations

## 🚀 Getting Started

### Prerequisites
- C++ compiler (GCC, Clang, or MSVC)
- Node.js (for JavaScript implementations)

### Running C++ Programs
```bash
g++ filename.cpp -o output
./output
```

### Running JavaScript Programs
```bash
node filename.js
```
